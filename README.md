
# Mario Bros 🧰

**Landing Page interativa para encanadores ** - Um projeto completo com formulário de contato integrado e design responsivo!

Este projeto foi desenvolvido durante o curso do **[DevClub](https://www.devclub.com.br)** como uma landing page completa para os encanadores Mario e Luigi. Através deste projeto, pude praticar manipulação do DOM, integração com serviços de formulário, animações CSS e design responsivo para diferentes tamanhos de tela.

## 📖 Sobre o Projeto

O **Mario Bros Encanadores** é uma landing page desenvolvida para apresentar os serviços dos encanadores mais queridos do mundo dos games. O projeto conta com um formulário de contato totalmente funcional integrado com a API do **FormCarry**, permitindo que os clientes enviem suas solicitações de orçamento diretamente. O design inclui um vídeo de fundo com máscara gradiente, animações suaves nos botões, links para WhatsApp e total responsividade para dispositivos móveis.

## ✨ Funcionalidades Principais

- ✅ Formulário de contato funcional integrado com API FormCarry
- ✅ Botão com link direto para WhatsApp
- ✅ Vídeo de fundo com máscara gradiente 
- ✅ Design responsivo (se adapta a celulares e tablets)
- ✅ Formulário com transição lateral animada e máscara de fundo

## 🎯 Objetivos do Projeto

- ✅ Aprender a integrar APIs de formulário (FormCarry)
- ✅ Praticar manipulação do DOM com JavaScript (abrir/fechar formulário)
- ✅ Desenvolver animações CSS com transitions e transforms
- ✅ Implementar design responsivo com media queries
- ✅ Criar uma interface profissional com vídeo de fundo
